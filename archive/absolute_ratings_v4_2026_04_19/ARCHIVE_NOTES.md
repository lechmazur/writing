# Absolute-Rating Archive

This directory preserves the previous V4 README publication surface from before the benchmark switched its public quality ranking to pairwise comparison ratings.

Archived here:
- `README.md`: the old absolute 0-10 rubric-rating README.
- `images/`: local archive of the old README-linked and related absolute-rating chart PNGs.

The active root README now uses the inter-LLM comparison Thurstone rating as the canonical leaderboard. Absolute rubric means, normalized absolute-score charts, and grader-vs-model heatmaps are retained for audit/debug context only.

PNG files are ignored by this repo's normal git policy, so the image archive is local workspace state unless explicitly packaged for publication.
